# Farmer's Productivity Optimization

This is a web application designed to optimize productivity of farmers by predicting the appropriate crop to be grown based on the soil and weather conditions.

# Dependencies
Flask==2.1.0

numpy==1.19.5

# Getting started

Clone the repository.

Install the dependencies using pip install -r requirements.txt.

Run the application using python app.py.

Navigate to http://localhost:5000 in your web browser to use the application.


# Usage

Enter the soil and weather conditions for your farm in the input form.

Click the 'Predict' button to see the recommended crop.

The application will display the recommended crop based on the input conditions.


# About the model

The model used for this application is a machine learning model trained on a dataset of soil and weather conditions and the corresponding crop yield. The model uses the input conditions to predict the appropriate crop to be grown to optimize productivity.
